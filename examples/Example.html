PyMdown Example

PyMdown is CLI tool to convert or even batch convert markdown files to HTML. It is written in Python and uses Python Markdown as the markdown parser and Pygments for code syntax highlighting. It supports all of Python Markdown's built in extensions. PyMdown also provides a number of additional extensions as well.

Example Syntax

test: This example of normal meta extension title: This title will be overridden by YAML

Extensions Used in this Document

Notes

  • sane_lists will alter the results of Mixed Lists. When turned off, this document will have all list items mixed and aligned proper. With sane_lists on, some will not be recognized, and some items may be aligned in different lists. sane_lists is disabled in this example.
  • having guess_lang=False allows selective highlighting of only the blocks that specify a language. When omitted or set true, it can be expected that all of the blocks will be highlighted to some extent (in some cases very wrong). guess_lang is disabled in this example.
  • pymdown.b64 can be verified by checking the source of one of the images to see if conversion occured.
